## Proyecto

Durante la actividad física intensa, el cuerpo humano experimenta un aumento en la temperatura corporal y en la frecuencia cardíaca debido al incremento del metabolismo. Cuando estas variables superan ciertos límites, pueden aparecer condiciones de estrés térmico que afectan el rendimiento del deportista y, en casos extremos, pueden generar riesgos para la salud como deshidratación o golpes de calor. La detección temprana de estas condiciones resulta importante para prevenir situaciones de sobrecalentamiento durante la actividad física. 

Y aunque actualmente existen dispositivos que permiten monitorear variables fisiológicas como la frecuencia cardíaca o la temperatura corporal, muchos de estos sistemas se limitan a mostrar información al usuario sin incorporar mecanismos automáticos de respuesta que ayuden a reducir el riesgo asociado a estas condiciones. 

En este contexto, el presente proyecto propone el desarrollo de un sistema embebido capaz de monitorear en tiempo real la temperatura superficial de la piel y la frecuencia cardíaca de un deportista durante la actividad física. A partir de estas mediciones, el sistema evaluará el estado fisiológico del usuario y determinará si se presentan condiciones asociadas a un posible sobrecalentamiento. Cuando dichas condiciones sean detectadas, el sistema activará un mecanismo de enfriamiento mediante una microbomba encargada de hacer circular líquido refrigerante a través de un sistema de enfriamiento portátil. Adicionalmente, el sistema registrará eventos relevantes de funcionamiento y permitirá visualizar el estado del sistema mediante una interfaz de usuario. 

El desarrollo del proyecto incluye el diseño de la arquitectura de hardware y firmware, la integración de sensores fisiológicos, la implementación del control del actuador de enfriamiento y la validación del prototipo. El sistema se plantea como un prototipo experimental orientado a demostrar el funcionamiento de un sistema embebido aplicado al monitoreo fisiológico y la prevención de sobrecalentamiento durante la actividad física.
